# Online Courses Platform

Интерактивная платформа для отображения онлайн-курсов с фильтрацией по категориям и поиском.

## Описание

Современный веб-интерфейс для каталога онлайн-курсов с возможностью фильтрации по категориям, поиска по названию и имени преподавателя, а также адаптивным дизайном для всех устройств.

## Функциональность

- ️**Фильтрация по категориям** - Marketing, Management, HR & Recruiting, Design, Development
- **Поиск** - мгновенный поиск по названию курса и имени преподавателя
- **Адаптивный дизайн** - корректное отображение на мобильных устройствах, планшетах и десктопах
- **Анимации** - плавные переходы и эффекты при наведении на карточки
- **Карточки курсов** - изображение, категория, название, цена и информация о преподавателе
- **Кнопка "Load more"** - имитация подгрузки дополнительных курсов

## ️ Технологии

- **HTML5** - семантическая разметка
- **CSS3** - стилизация, Grid Layout, Flexbox, анимации
- **JavaScript (ES6+)** - динамическая фильтрация и поиск
